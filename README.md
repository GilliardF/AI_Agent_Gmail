# AI_Agent_Gmail
Agente de IA em Python para automação inteligente do Gmail. Integrado com a API Gemini do Google para ler, interpretar e gerar respostas contextuais. Automatiza o gerenciamento da sua caixa de entrada, organizando e respondendo e-mails de forma autônoma. Conexão segura via API oficial do Gmail para uma gestão eficiente.
